## Instrukcja publikacji manualnej

1. Uruchom: `python3 cli.py open-product <slug>`
2. Wklej tytuł, opis, cenę na etsy.com/your/listings/create
3. Dodaj tagi (max 13)
4. Uploaduj renders/ w kolejności: hero, lifestyle, sizes, detail, info
5. Ustaw profil wysyłki UE i kliknij Publish

> Gdy Etsy API zostanie zatwierdzone: `python3 cli.py etsy-auth` + `python3 cli.py publish <slug>`
